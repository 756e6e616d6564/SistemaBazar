# Dependencias del proyecto SistemaBazar

## CSS y JS (CDN)
- Bootstrap CSS v5.3.0: https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css
- Bootstrap Bundle JS v5.3.0: https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js
- Bootstrap Icons v1.11.0: https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.0/font/bootstrap-icons.css

> Nota: Algunas vistas usan Bootstrap Icons v1.8.1; se recomienda unificar a la versión 1.11.0.

## PHP
- Extensión PDO (uso de pdo_helpers)

## JavaScript
- No hay librerías externas adicionales detectadas aparte de Bootstrap.

## Consideraciones
- Los enlaces de CDN se están usando directamente en vistas. Se puede optar por descargar dependencias localmente o usar un gestor de paquetes.
